- Syntax:
	- First, set mode (modes change the parsing method, simple code versus raw logical statements.)
	- Define inputs and output at start (lowercase, can set so that it renders as uppercase).
	```gates
	set 0;
	in a,b;
	out q^;
	a,b>A>q;
	```
